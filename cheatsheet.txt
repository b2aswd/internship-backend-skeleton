1. Instalace vagrant pluginů
1.1 vagrant plugin install vagrant-vbguest
1.2 vagrant plugin install vagrant-disksize
1.3 Pouze pro Windows: vagrant plugin install vagrant-winnfsd
2. vagrant up
2.1 Pokud vyhodí chybu s grub-pc (převážně Windows), tak vagrant ssh a uvnitř spustit DEBIAN_FRONTEND=dialog dpkg --configure grub-pc
3. vagrant ssh
4. cd /var/www/projects
5. git clone https://github.com/b2aswd/internship-backend-skeleton.git
6. cd internship-backend-skeleton
7. docker-compose -p simple_messenger -f docker/docker-compose.yml up -d
8. docker exec -it simple_messenger_backend_1 bash
9. composer install
10. Připojení k databázi v DataGrip
11. Aplikování SQL scriptu (vytvoření databáze a tabulky pro uživatele) přes DataGrip
12. Otevření projektu v PhpStorm
13. Nastavení PHP verzi na 7.4 v PhpStorm
14. Nastavení Xdebug (nástroj pro debugování kódu)
